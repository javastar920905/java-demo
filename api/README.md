# 项目说明
提供第三方通用服务接口 短信 邮件 支付 生成二维码等
其他项目如需用到该项目的工具类,直接引入该项目即可.


# 环境说明
* JDK版本为 : 1.8.0_*
* GRADLE版本为 : 3.4.*
